> ! These files will only work on the latest [awesome](https://github.com/awesomeWM/awesome) git master branch.

## File Structure

- :rocket: `apps.lua`
    - I define often used apps here so i can easily launch them later.
- :star: [bling](https://github.com/Nooo37/bling)
    - Utilities for Awesome.
- :candy: `candy`
    - Visual stuff like bars, titlebars, notifications and panels.
- :sparkling_heart: `helpers.lua`
    - Functions i use often defined here.
- :white_heart: `icons`
    - Collection of icons, for widgets.
- :heart: `keys.lua`
    - A lot of keybinds.
- :black_heart: `module`
    - More visual stuff like application launcher, dock, lockscreen.
- :rainbow: `rc.lua`
    - Where it all comes together.
- :shit: `shit`
    - Shitty daemons.
- :first_quarter_moon: `themes`
    - `Beautiful` themes.
- :zap: `widgets`
    - Widgets used more then once.
