* Bin

These are my shell scripts, that makes my life easier.

You should edit them to match your setup.

| name  | description                      | usage                |
|       |                                  |                      |
| arc   | archives a folder                | arc <folder>         |
| art   | extracts cover from music        | art <music>          |
| clone | easily clone a repo              | clone <user/repo>    |
| conv  | convert every jpg to png         | run conv in a folder |
| cx    | toggle chmod +x on a file        | cx <file>            |
| dl    | download from my most used sites | dl <url>             |
| ext   | extracts an archive              | ext <archive>        |
| lock  | locks screen                     | run lock             |
| push  | push a repository                | run push             |
| rec   | records the screen               | run rec              |
| shoot | take a screenshot                | run shoot            |
| upl   | upload a text file for 0x0.st    | upl <file>           |
| chth  | change theme 					   | run chth		      |
